# 智慧医疗平台管理系统
智慧医疗平台管理系统 功能: 
管理员端: 有用户管理、医生管理、科室分类管理、科室信息管理、在线挂号管理、预约体检管理、体检报告管理、药品类型管理、药品信息管理、处方信息管理、病历信息管理、智能导诊管理、导航服务管理、就诊卡充值管理、系统管理 
用户端: 科室信息、导航服务、公告信息、后台管理、在线问诊功能
